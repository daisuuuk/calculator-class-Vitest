## 概要

電卓アプリ

※本アプリでの対応は実装・テストのみ

## 要求整理

### 必須要件

 - 整数および小数の四則演算（+, -, ×, ÷）ができる

### 制約条件

 - ユーザー入力は8桁に制限
 - 計算結果が8桁を超える場合、指数表記にする
 - JavaScript 標準の `Number` 型を使用
 - 左から順に計算（通常の算術優先順位は考慮しない）             
 - 0除算のような計算をした場合、計算結果はエラーとなる

### 曖昧だった点と解釈

 - ２つ目の数字の負の符号の扱い方 → 2つ目の数字では負の符号を扱わない
 - 履歴エリアの仕様追加 → 定期的な仕様書確認の必要性
 - 「0.」補完にするかどうか → 「0.」補完が必要
 - 「=」押下後の数字入力反映 → 新しい計算を始める
